# 交换两个变量的值的方法

1. 值引用（这种不可行）

   ```c
   void swap(int a, int b){
       int temp;
       temp = a;
       a = b;
       b = temp;
   }
   swap(a, b)
   ```

2. 指针引用（这种是传统的 C 语言的写法）

   ```c
   void swap(int *a, int *b){
       int temp = *a;
       *a = *b;
       *b = temp;
   }
   swap(&a, &b);
   ```

3. C 加加的引用

   ```cpp
   void swap(int& a, int& b){
       int temp = a;
       a = b;
       b = temp;
   }
   swap(a, b)
   ```
